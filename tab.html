HTML is een taal om de structuur van documenten te beschrijven. Een webbrowser zet HTML-code om in een simpele grafische weergave voor de gebruiker. Als je de opmaak wilt aanpassen, heb je CSS nodig en met Javascript kunt je de pagina volledig interactief maken.

Met CSS kun je de opmaak van een HTML-documenten aanpassen. Denk aan lettertypes, kleuren, marges, kaders en nog veel meer.

Javascript is een scripttaal die voornamelijk gebruikt wordt op webpagina's. Je kunt er allerlei interessante effecten mee programmeren door on-the-fly CSS en HTML aan te passen.
